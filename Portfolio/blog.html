How to start Data Struture & Algorithm (DSA) as Begineers

Learning Dsa its become hard if you do not follow right path as begineers , if you starts good then you can finish it good

Steps You have to follow for learn DSA
  • Start with any langauge (Prefered c++)
  • Learn all basics of langauge as
    • Basics syntax
    • Operators
    • conditional statements
    • Loops
    • Array and Strings
    • STL
  • Algorithums
    • Searching algorithms
      • Linear search
      • Binary Search
    • Sorting algorithms
      • Selection Sort
      • Bubble Sort
      • Insertion Sort
      • Merge Sort
      • Quick Sort
      • Count Sort
      • Vector sort
  • Learn Recursion
  • Now You can start with learning acutal data structure topics
    • Arrays
    • String
    • Linked list
    • Stack
    • Queue
    • Tree
    • Segment Tree
    • Graph
  • Learn paradiams
    • Recursion
    • Backtracking
    • Dynamic programming (DP)
    • Greedy Alogorithm
